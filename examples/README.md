# Otis AI Platform Examples

This directory contains practical examples and use cases for the Otis AI Platform, demonstrating both cybersecurity and anti-spam capabilities.

## Directory Structure

```
examples/
├── anti_spam/
│   ├── basic_classification.py          # Simple spam detection
│   ├── batch_processing.py              # Process multiple emails
│   ├── security_wrapped_model.py        # Classification with security
│   └── adversarial_testing.py           # Test model robustness
├── red_team/
│   ├── attack_vectors.py               # Individual attack examples
│   ├── multi_turn_orchestration.py     # Advanced attack chaining
│   └── model_robustness.py             # Comprehensive testing
├── blue_team/
│   ├── threat_detection.py             # Real-time detection
│   ├── automated_remediation.py        # Response automation
│   └── compliance_monitoring.py        # NIST RMF compliance
└── integration/
    ├── api_usage.py                    # REST API examples
    └── docker_deployment.py            # Container deployment
```
